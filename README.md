# Sistem Informasi Manajemen Kos

## Permasalahan

### 1. Pembayaran masih manual
- Penghuni membayar secara cash / transfer tanpa sistem
- Sering terjadi keterlambatan pembayaran
- Pemilik kesulitan mengecek status pembayaran

### 2. Tidak ada sistem pencatatan yang rapi
- Data penghuni dicatat melalui buku atau chat
- Risiko kehilangan data
- Sulit mencari data lama

### 3. Keluhan tidak terorganisir
- Penghuni melapor melalui chat pribadi
- Laporan sering terlewat
- Tidak ada status penanganan

### 4. Kurangnya komunikasi terpusat
- Informasi penting disampaikan manual
- Tidak semua penghuni menerima informasi

### 5. Pengelolaan kamar kurang efisien
- Tidak ada sistem monitoring kamar
- Sulit mengetahui kamar kosong atau terisi

### 6. Tidak ada notifikasi / reminder
- Penghuni sering lupa membayar
- Tidak ada pengingat otomatis
